# GREEN API TEST
Инструкция:

Клонируем репозиторий к себе на на компьютер.
Для начала надо сконфигурировать .env файл
```code 
DATABASE_URL=<db url>
NEXTAUTH_JWT_SECRET=<secret>
NEXTAUTH_SECRET=<secret>
```

Устанавливаем зависимости
```code
npm install
```

Запускаем дев сервер
```code
npm run dev
```
